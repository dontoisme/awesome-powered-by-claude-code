## Customization

Want to modify the badges? The SVG source files are in this directory. Key colors:

| Element | Color | Hex |
|---------|-------|-----|
| Anthropic Orange | Terracotta | `#D4956A` |
| Dark Background | Deep Purple | `#1A1525` |
| Gradient Start | Muted Purple | `#2D2640` |
| Text | White | `#FFFFFF` |
